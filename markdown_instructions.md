## Списки

1. Чтобы добавить ненумерованые списки, необходимо пункты выделить звездочкой(*), например:
* Элемент 1;
* Элемент 2;
* Элемент 3;

Чтобы добавить нумерованные списки, необходимо пункты просто пронумеровать, например:
1. Элемент 1;
2. Элемент 2;
3. Элемент 3;
